## 1. 시작하기

### 1-1. 회원가입 및 로그인

| 경로 | 설명 |
|------|------|
| `/login` | 로그인/회원가입 페이지 |

- 이메일 + 비밀번호로 회원가입
- 비밀번호 확인란 + 보기/숨기기 토글
- 가입 즉시 **7일 Premium 무료 체험** 자동 활성화
- 이메일 인증 필요 (Supabase Auth)

### 1-2. 온보딩 마법사 (5단계)

| 단계 | 화면 | 입력 | 설명 |
|------|------|------|------|
| 1/5 | 사이트 연결 | WordPress URL, 사용자명, 앱 비밀번호 | 연결 테스트 버튼으로 검증. URL에 `https://` 자동 보정 |
| 2/5 | 카테고리 선택 | 최소 2개 선택 | Standard 19개 카테고리 중 선택 |
| 3/5 | 발행 스케줄 | 프리셋 선택 | 매일 2회 / 매일 4회 / 평일만 |
| 4/5 | 블로그 단계 | 라디오 선택 | 처음 시작 / AdSense 승인 전 / 승인 완료 / 수익화 진행 중 |
| 5/5 | 완료 | - | 설정 요약 확인 후 대시보드로 이동 |

**블로그 단계별 자동 전략 설정:**

| 선택 | 수익화 Stage | 엔진 전략 |
|------|------------|----------|
| 처음 시작 | Stage 1 | 순수 정보성 글 발행 (AdSense 승인 목표) |
| AdSense 승인 전 | Stage 1 | 품질 높은 글 발행 (승인 조건 충족) |
| AdSense 승인 완료 | Stage 2 | AdSense + 텐핑 CPA 수익화 시작 |
| 수익화 진행 중 | Stage 3 | 모든 수익 채널 자동화 (쿠팡 API + 텐핑 + 레퍼럴) |

## 2. 홈 대시보드

| 경로 | 설명 |
|------|------|
| `/dashboard` | 소비자 메인 화면 |

### 2-1. 스코어카드 (4개)

| 카드 | 데이터 소스 | 부가 정보 |
|------|-----------|----------|
| 오늘 발행 | `publish_logs` (당일) | 목표 편수 표시 |
| 이달 수익 | `revenue` (이번 달) | vs 지난달 증감률 |
| 총 발행 | `publish_logs` (전체 success) | - |
| 블로그 건강도 | 자동 계산 (50~100점) | 양호/보통/주의 |

**건강도 점수 계산:**
- 기본 50점
- 총 글 30편+: +20점 / 10편+: +10점
- 오늘 발행 2편+: +15점
- 실패 0건: +10점
- 수익 발생: +5점

### 2-2. 수익 트렌드 차트
- 최근 30일 일별 수익 AreaChart
- 데이터 없으면 안내 메시지 표시

### 2-3. 다음 할 일 (Smart Actions)
현재 상태에 따라 자동 추천:

| 조건 | 추천 액션 |
|------|----------|
| 총 글 < 20편 | "AdSense 승인까지 X편 더 필요합니다" |
| Stage 1 & 글 20편+ | "AdSense 승인을 신청할 준비가 됐습니다!" |
| Stage 2 | "쿠팡 파트너스 상품을 등록하면 수익이 증가합니다" |
| Stage 3 | "모든 수익 채널이 가동 중입니다" |

### 2-4. 최근 발행 (5건)
- 제목, 발행일, 카테고리, 품질 점수 Badge
- 품질 색상: 90+ 초록, 80+ 노랑, 80 미만 빨강

### 2-5. 마일스톤 (상위 5개)

| 마일스톤 | 목표 | 측정 기준 |
|---------|------|----------|
| 첫 글 발행 | 1편 | total_posts |
| 10편 달성 | 10편 | total_posts |
| 30편 달성 | 30편 | total_posts |
| AdSense 승인 | 승인 | adsense_approved |
| 첫 수익 달성 | 수익 발생 | first_revenue |

- ProgressBar로 진행률 표시
- 달성 시 체크마크 전환

### 2-6. Premium 업그레이드 배너
- Standard 플랜일 때만 하단에 표시
- "Golden Mode, 고급 분석, 일 20편 발행, 맞춤 스케줄"

## 3. 내 블로그

| 경로 | 설명 |
|------|------|
| `/blog` | 발행 글 관리 |

### 3-1. 필터 & 검색
- 기간 필터: 전체 / 오늘 / 이번주
- 제목 검색

### 3-2. 발행 목록
- 최대 50건 표시
- 항목: 제목, 카테고리, 발행일, 글자수, 품질 점수
- WordPress 외부 링크 (새 탭)

### 3-3. 카테고리 분포
- PieChart (상위 5개 범례)

### 3-4. 성과 분석 [Premium]
- Standard: PlanLock 오버레이로 잠금
- Premium: 글별 예상 수익, SEO 점수 (향후 구현)

## 4. 수익

| 경로 | 설명 |
|------|------|
| `/revenue` | 수익 현황 및 로드맵 |

### 4-1. 채널별 수익 카드
- 총 수익 + 상위 3개 채널 (AdSense, 쿠팡, 텐핑)
- 이번 달 합계 + vs 지난달 증감률

### 4-2. 수익 추이 차트
- 최근 30일 AreaChart
- 일별 합계

### 4-3. 수익화 로드맵 (3단계)

| Stage | 이름 | 핵심 전략 | 팁 |
|-------|------|----------|-----|
| 1 | AdSense 승인 | 20편+ 고유 콘텐츠, 필수 페이지 | About/Privacy/Contact 필수, 제휴 링크 금지 |
| 2 | 수익화 시작 | 텐핑 CPA + 쿠팡 수동 링크 | 고단가 오퍼 선택, 쿠팡 15만원 판매 목표 |
| 3 | 수익 극대화 | 쿠팡 API + 텐핑 풀가동 + 레퍼럴 | 딥링크 자동화, RPM 높은 카테고리 확대 |

- 현재 단계: 색상 하이라이트 + "현재" Badge
- 완료 단계: 초록 체크마크 + "완료" Badge

### 4-4. 수익 시뮬레이션 [Premium]
- Standard: PlanLock 잠금
- Premium: 현재 속도 기반 예측 (향후 구현)

## 5. 설정

| 경로 | 설명 |
|------|------|
| `/settings` | 사이트 및 발행 설정 |

### 5-1. 내 정보
- 이름 변경
- 이메일 (읽기 전용)
- 현재 플랜 표시 + 업그레이드 링크

### 5-2. 내 사이트
- 연결된 WordPress 사이트 표시
- 연결 상태 확인
- Premium/MaMa: 사이트 추가 가능 (최대 3개/무제한)

### 5-3. 발행 스케줄

| 프리셋 | 시간 | 플랜 |
|--------|------|------|
| 매일 2회 | 08:00, 18:00 | 전체 |
| 매일 4회 | 07/12/17/22시 | 전체 |
| 평일만 | 평일 08:00 | 전체 |
| 직접 설정 | 커스텀 | Premium+ |

### 5-4. 카테고리 관리

**Standard (19개 사용 가능)**

| 그룹 | 카테고리 |
|------|---------|
| 제품 리뷰/비교 | AI 도구, IT/전자기기, 스마트홈, 반려동물, 건강/웰니스, 재테크, 뷰티, 육아/유아, 생활가전, 운동기구, 교육/생산성 |
| 정보 서비스 | 정부지원/보조금, 세무/절세, 여행 정보, 기관 정보, 행사/컨퍼런스 |
| 홍보/마케팅 | 니치 홍보용, 브랜드 콘텐츠, 비교 랜딩 |

**Premium/MaMa 추가 (12개)**

| 그룹 | 카테고리 |
|------|---------|
| 뉴스/리서치 | SBS 뉴스, KBS 뉴스, JTBC 뉴스, SNS 인기 이슈, 10대 대기업 |
| 섹터 리서치 | 반도체, AI/인공지능, 방산, 제약/바이오, 로봇, 전기차/2차전지, 우주/항공 |

### 5-5. 초기 설정 (원클릭)

| 순서 | 버튼 | 워크플로우 | 결과 |
|------|------|-----------|------|
| 1 | 메뉴 자동 설정 | `setup-menu.yml` | About/Privacy/Contact 페이지 + 네비게이션 메뉴 생성 |
| 2 | CSS 디자인 적용 | `inject-css.yml` | 블로그 테마 맞춤 스타일링 자동 적용 |
| 3 | 첫 글 발행 (3편) | `publish.yml` (count=3) | AI가 3편 자동 작성 및 발행 |

- 각 버튼 클릭 시 GitHub Actions 백그라운드 실행
- 상태: 실행 중 → 완료/실패 실시간 표시
- 추천 순서: 메뉴 → CSS → 글 발행

### 5-6. 로그아웃
- 세션 종료 후 `/login`으로 이동

## 라우팅 구조

```
/login          → 로그인/회원가입
/onboarding     → 5단계 초기 설정
/dashboard      → 소비자 홈
/blog           → 발행 글 관리
/revenue        → 수익 현황
/settings       → 설정
/upgrade        → 플랜 업그레이드
/               → 관리자 대시보드 (기존, admin만 사이드바 링크)
```
